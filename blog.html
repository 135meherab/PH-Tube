var is a functional Scope. That means the variable can be accessable from any where in the fucntion where it was diclared.
let & const are block Scope. That means it will work only in that block where it was diclared.

Both null and undefined are used to denote that a variable has no value.
The difference is that the null value is assigned to the variable while declaring the variable.
And in case of undefined, no value is assigned to the variable, it is only declared.
